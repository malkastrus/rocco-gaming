# SharkByte - split completo

Este proyecto es el corte más seguro del último HTML completo.

## Estructura
- `index.html`: web completa tal como estaba en el HTML original.
- `css/base.css`: estilos principales y visual completa.
- `css/admin-filtros.css`: mejoras visuales del panel admin en filtros.
- `css/badges-y-toggles.css`: badges, toggles y check visual.
- `js/app.js`: lógica principal de la tienda y del panel.
- `js/admin-mejoras.js`: mejoras extra del admin y features nuevas.
- `assets/`: carpetas listas para meter imágenes, iconos y favicon.
- `docs/`: notas e instrucciones.

## Cómo subirlo a GitHub
Sube la carpeta completa respetando la jerarquía.

## Cómo abrirlo localmente
Haz doble click en `index.html` o usa una extensión tipo Live Server.

## Nota
No separé aún `admin.html` para no arriesgar romper la lógica del archivo original.
Este split conserva la visual y funcionalidad del HTML base.
